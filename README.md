# CV Builder Application

### A modern application for creating professional resumes

### Technology Stack
- React
- JavaScript
- TailwindCSS
- PostCSS
- HTML5
- CSS3
- Git
- Vite

### Features
- Dynamic resume generation
- Customizable sections:
  - Photo upload
  - Contact information
  - Work experience
  - Education
  - Professional summary

### Installation
`npm install`

### Development Server
`npm run dev`

### Build Project
`npm run build`
